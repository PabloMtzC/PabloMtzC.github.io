Cuestionario

1.¿Qué es la ciberseguridad?

conjunto de prácticas, tecnologías y medidas diseñadas para proteger los sistemas informáticos y los datos de posibles ataques o accesos no autorizados.
area de la informatica enfocada en el desarrollo de programas maliciosos
practicas y desarrollo tecnologico enfocado en la arquitectura de software libre
desarrollo de sistemas operativos seguros

2.¿Qué es un ataque de phishing?

ataque cibernetico donde el atacante sobrecarga el servidor provocando así una denegación del servicio en Pishing
envío y descarga de un malware que obtiene los datos personales del atacado
ataque cibernético en el que los delincuentes promueben información falsa de un usuario
ataque cibernético en el que los delincuentes se hacen pasar por entidades confiables para engañar a las personas y obtener información

3¿Qué es un ransomware?

un troyano que corormpe la información del computador del atacado
tipo de malware que bloquea el acceso a los archivos o sistemas de una víctima y exige un rescate
es un método por el cual los atacantes se hacen pasar por el usuario y lo chantajean con información privada
tipo de software legítimo y beneficioso que se utiliza para proteger los archivos y sistemas de los usuarios

4.¿Qué es un firewall?

dispositivo que se utiliza para calentar y generar fuego en entornos controlados
barrera de seguridad que controla y regula el tráfico de red entre una red privada y una red externa
software que se utiliza como antivirus y protección del usuario en internet
herramienta utilizada para la detección y rastreo de hackers o crakers en internet

5.¿Qué es una vulnerabilidad de seguridad?

es la protección total de un sistema informático
es una sección de un sistema informatico creado especificamente para asegurarse que el sistema está completamente seguro
debilidad en un sistema informático o una aplicación que podría ser explotada por atacantes
facilidad de un gestor de busqueda para acceder al sistema o aplicación y encontrar contraseñas o material de seguridad informatico

6.¿Qué es el cifrado de datos?

proceso de codificar la información para que solo las personas autorizadas puedan acceder y comprender su contenido
es el proceso de guardar la información para que solo personas con acceso publico puedan acceder y comprender su contenido
codificar información para posterior exponer el código
añadir ñ a las contraseñas para que hackers extrangeros no puedan reescribirlas

7.¿Qué es la ingeniería social?

Ingeniería aplicada a la construcción y arquitectura en grandes grupos y areas publicas así como parques, ciudades, etc.
tecnica de engaño donde los atacantes provocan ataques directos a sistemas informaticos robando información
Ingeniería aplicada en la manipulación de redes sociales
técnica en la que los atacantes manipulan y engañan a las personas para obtener información confidencial

8.¿Qué es un virus informático?

es una aplicación generada por la computadora cuando no cuenta con un sistema de seguridad o antivirus
un programa benefico diseñado para generar anticuerpos en la computadora y mejorar su sistema operativo
programa malicioso diseñado para infectar archivos o sistemas informáticos
una enfermedad común en los sistemas informaticos cuando no cuentan con un software antivirus

9.¿Qué es malware?

cualquier software malicioso diseñado para dañar o infiltrarse en un sistema informático sin el consentimiento del usuario.
cualquier software malicioso creado por el mismo usuario cuando borra información importante
un sistema operativo malicioso diseñado para dañar o infiltrarse en un sistema informático sin el consentimiento
cualquier software benefico diseñado para proteger la información del usuario

10.¿Qué es un ataque de denegación de servicio (DDoS)?

intento de denegar el sevicio a un atacante en el servidor puesto que busca dañarlo
intento de sobrecargar un servidor, sistema o red provocando una interrupción del servicio para los usuarios legítimos.
(Distributed Denial of Service) es una técnica de jardinería que consiste en plantar una variedad de flores distribuidas
ataque provocado al usuario cuando los atacantes obtienen la información de este y cambian su contraseña denegando el servicio

11.¿Qué es un análisis de vulnerabilidades?

analisis del diseñador del sistema donde evalua que el diseño de la pagina no cuente con un error
analisis del progrador donde verifica que el sistema cuente con debilidades para postergar su parche
proceso que identifica y evalúa las debilidades de seguridad en un sistema o red
verificación de errores en el computador del programador

12.¿Qué es el acceso no autorizado?

salida del sisema operativo posterior a su uso y niega el servicio
entrada o uso del interior a la propiedad sin permiso o autorización por parte del propietario
entrada o uso de un sistema informático, red o dispositivo sin permiso o autorización por parte del propietario o administrador.
negación del servicio DDoS por uso del servidor

13.¿En ciberseguridad, qué es la ingeniería inversa?

proceso de construcción y analisis de software aprendiendo a usarlo en el proceso
